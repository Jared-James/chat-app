# 🏣 chat-app 

A simple chat room application created with sockets & express.

---

### Users can 
- 🚩 Create a user 
- ⏩ Create a room 
- 🕐 Enter a room 
- 📞 send messages  
- ⏰ Be notified when a user enters or exits a room 
- 🌎 send their location  

---
Install
- Clone repo
- NPM install

Scripts
- NPM RUN start


---

+ ✔️  Language: Javascript
+ ✔️  Server: Express
+ ✔️ Style: CSS

---

- localhost:3000
